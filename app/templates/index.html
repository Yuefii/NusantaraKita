Introduction

Nusantara Kita adalah API yang menyediakan data wilayah Indonesia untuk berbagai aplikasi yang memerlukan informasi geospasial. Project ini adalah pembaruan dari versi sebelumnya, dengan peningkatan signifikan dalam cara penyimpanan dan pengelolaan data.

Sebelumnya, data wilayah dihasilkan dari file CSV, namun kini Nusantara Kita menggunakan database modern untuk meningkatkan performa dan fleksibilitas. Anda sekarang dapat memilih dari tiga opsi database yang didukung: MySQL, PostgreSQL, dan MongoDB. Dengan perubahan ini, API tidak hanya menjadi lebih efisien tetapi juga lebih mudah diintegrasikan ke dalam berbagai sistem.

Versions:

API Version 2.0

Installation

bagaimana menjalankan projectnya dilokal?

  1. Clone Repository:
    git clone https://github.com/username/NusantaraKita.git
    cd NusantaraKita

  2. Setup Environment:
    mv env.example .env
    lalu kemudian kalian isi sesuai database kalian.

  3. Install Requirements:
    pip install -r requirements.txt

  4. Menjalankan Jupyter Notebook nya:
    jupyter notebook
    jalankan semua code yang ada di notebook untuk merubah dataset csv menjadi sql yang akan langsung diinsert ke database kalian.

  5. Menjalankan API nya:
    python main.py
